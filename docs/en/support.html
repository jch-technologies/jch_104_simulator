How do I connect to a PLC?

1. Switch to Client mode
2. Enter the IP address and port (default: 2404) of your PLC
3. Click "Connect"
4. After successful TCP connection, click "STARTDT" to start data transfer

How do I test Server mode?

1. Start the server on your iPad/Mac
2. Load demo data points
3. Connect with a second device in Client mode or a PLC/SCADA system

What does "Select before Execute" mean?

When S/E is enabled, a SELECT command is sent first. The remote station must confirm this before the actual EXECUTE command is sent. This is a safety feature according to IEC 60870-5-104.
